- 初始化npm，生成package.json文件
```
 npm init -y
```
命令的作用是: 对项目进行初始化操作,对包进行管理

-y 的含义:yes的意思,在init的时候省去了敲回车的步骤,生成的默认的package.json

- 安装mathjax
npm install mathjax@3
- 查看安装结果 
- npm ls
